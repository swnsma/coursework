HTML(Hyper Text Markup Language) - язык гипертекстовой разметки. Был создан в 1989 году. Команды HTML называются тегами. Теги делятся на два вида: одиночные и парные. За этапы своего развития язык пережил множество изменений. На даный момент последняя версяя языка - HTML 5.0. XHTML(eXtensible Hyper Text Markup Language) - расширеный HTML. XHTML более требовательный к синтаксису, нежели HTML. Различия:

  1. В XHTML документе все теги должны закрываться.
  2. Названия тегов и параметров записываются исключительно в нижнем регистре.
  3. В XHTML должны выполняться правила вложения тегов
  4. Значение параметров тегов в XHTML записуются в двойных кавычках.
  5. Все параметры в XHTML должны иметь значения
  6. XHTML в первых строках должен иметь служебный тег ‹!DOCTYPE›
Самый простой XHTML имеет вид:
‹!DOCTYPE html PUBLIC " -//W3C//DTD XHTML 1.0 Transitional//EN " " http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd "› ‹html› ‹head› ‹title›Название документа‹/title› ‹/head› ‹body› ‹/body› ‹/html›

Типы XHTML документов:
Strict - разрешает использовать теги исключительно логического форматирования.(Логическое форматирование - выделение структурных элементов страницы).
Transitional - разрешает использовать теги и параметры связаные с физическим форматированием страницы(Физическое форматирование - форматирование отображения на странице).
Frameset - проект с использованием вставки других страниц(использование фреймворков). Считается устарелым подходом.
В языке HTML есть комментарии. Коментарий открывается при помощи написания ‹!-- комментарий --›
